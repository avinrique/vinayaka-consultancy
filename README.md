# Vinayaka Consultancy — Client College Pages

Static HTML pages for Vinayaka Education Consultancy, providing detailed college admission information, fee structures, placement data, and application guidance. Layout inspired by EduDunia's college profile pages.

## Pages

- **RV_College.html** — RV College of Engineering (RVCE), Bangalore
  - Complete admissions guide (KCET, COMEDK, Management quota)
  - Fee structure, eligibility criteria, and entrance exam details
  - Placement statistics and top recruiters
  - Course toggle buttons (B.E, M.Tech, MCA)
  - FAQ accordion with 8 questions
  - Sticky header + tab navigation with scroll spy
  - Sidebar: Lead capture form, related colleges, course pills, featured colleges, student reviews
  - Apply Now modal with full form (Name, Email, Phone, State, City, Course, Message)
  - Floating "Enquire Now" button (desktop) + bottom action bar (mobile)
  - Fully responsive (desktop, tablet, mobile)

## Tech Stack

- Vanilla HTML/CSS/JS (single-file, self-contained)
- [Poppins](https://fonts.google.com/specimen/Poppins) (Google Fonts)
- [Font Awesome 6](https://fontawesome.com/) (icons)
- No frameworks or build tools required

## Deployment

Each `.html` file is standalone and can be served directly. Upload to any static host or drop into the existing Vinayaka Consultancy web root.
